# trading-fabric docs

This folder documents the package-level operating model for `@veridex/trading-fabric`.

- [architecture.md](architecture.md) explains the runtime composition, package surfaces, and event flow.
- [threat-model.md](threat-model.md) covers prompt injection, tool poisoning, secrets, approval, and execution risks.
- [policy-cookbook.md](policy-cookbook.md) gives policy and approval recipes for local and CI use.
- [migration-from-tradingagents.md](migration-from-tradingagents.md) maps the Python TradingAgents workflow to this TypeScript package.

The README is the quickstart. These docs are the operator and maintainer view.
